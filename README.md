# MemeMaker

O MemeMaker é um web app que gera memes, construído com React e Styled-Component.

O app consome uma API que contém várias imagens para a geração dos memes.

O usuário pode escolher uma imagem, digitar os textos e prontinho: meme criado!

## Tecnologias
* [React](https://pt-br.reactjs.org/)
* [Styled-Components](https://styled-components.com/)
* API do [Imgflip](https://imgflip.com/api)
